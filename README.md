# MAGO Client for Real-time Speech Recognition

## MacOS에 설치하기

### 설치하기 전에

- python 3.10 이상 설치되어 있어야 합니다.
- pip 22.0.4 이상 설치되어 있어야 합니다.

## `mago-client` 다운 받기

```bash
git clone https://github.com/holamago/mago-client.git
cd mago-client
```

### 패키지 설치

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 실행하기

```bash
source venv/bin/activate
python client/realtime.py
```
